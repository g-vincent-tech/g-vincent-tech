### ⚡ Architectural Philosophy
*   **Design for Failure:** Assume networks will drop and machines will fail. Build software with explicit safety boundaries and smart fallback defaults.
*   **Measure First:** Do not optimize code blindly. Rely entirely on profiling metrics and tracking tools to prove where a performance bottleneck actually sits.
